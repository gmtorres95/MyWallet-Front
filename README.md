## How to run

### Install the application

```bash
# Clone this repository
$ git clone https://github.com/gmtorres95/MyWallet-Front

# Install the dependencies
$ npm i
```

### Compile the code

```bash
$ npm run build
```

### Run the application

```bash
$ npm start
```

The server will automatically start at localhost:3000/ on your favorite browser
